# Analyse de feedbacks clients

Classification automatique de feedbacks clients par sentiment (positif, mitigé, négatif).

**Objectif** : obtenir une vue d'ensemble des avis clients à partir d'un fichier CSV.

**Technologies** : Python, Pandas, OpenAI API, Streamlit

**Utilisation** : charger un fichier CSV, choisir la colonne à analyser, lancer l'analyse.

**Résultat** : nombre de commentaires par catégorie de sentiment.
